## Installation
To install the project on your local system, run the following on your terminal.
 - `git clone https://github.com/anshv-py/grape-ai.git`
 - `cd grape-ai`
 - `npm install --force`
 - `$env:NODE_OPTIONS = "--openssl-legacy-provider"`
 - `npm start`
